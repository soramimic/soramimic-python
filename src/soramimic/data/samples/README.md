# サンプル単語リスト

出典: [soramimic-wordlists](https://github.com/soramimic/soramimic-wordlists) (bc0b32a 時点)。
権利上の配慮から、事実データ系のリストのみを同梱している(キャラクター名・実在人名のリストは同梱しない)。
更新する場合は soramimic-wordlists 側を正としてコピーし直すこと。

各リストの元データと帰属:

- **nations.csv** — 国名(国連加盟国)。[mledoze/countries](https://github.com/mledoze/countries)
  (Open Database License 1.0) のデータを元に作成。
- **sekitsui.csv** — 脊椎動物の和名。[Wikidata](https://www.wikidata.org/) (CC0) のデータを元に作成。
- **stations.csv** — 駅名。[Wikidata](https://www.wikidata.org/)・[Wikipedia](https://ja.wikipedia.org/)
  (CC BY-SA 4.0) から抽出した事実データ(駅名・読み・所在地)。画像列のURLは
  Wikimedia Commons を指し、利用時は image_page のクレジット条件に従うこと。
